适合谁看？

Stable Diffusion 是最强大的开源 AI 图像生成模型，支持本地部署、完全免费，无内容审查。通过 AUTOMATIC1111 WebUI 或 ComfyUI 即可快速上手，配合 LoRA 模型可生成各种风格的图像。

步骤 1

硬件要求

推荐：NVIDIA GPU（显存 ≥6GB），如 RTX 3060/4060。CPU 模式极慢，不推荐。无合适显卡可选择云端方案：Google Colab 或 RunPod。

截图预留位置：Stable Diffusion - 硬件要求
步骤 2

安装 AUTOMATIC1111 WebUI

前往 GitHub 搜索 AUTOMATIC1111/stable-diffusion-webui，按 README 说明安装 Python 3.10 + Git，运行 webui-user.bat（Windows）或 webui.sh（Mac/Linux）一键启动。

截图预留位置：Stable Diffusion - 安装 AUTOMATIC1111 WebUI
步骤 3

下载基础模型

推荐模型下载站：Civitai.com 和 HuggingFace。常用模型：真实系选 Realistic Vision，动漫风选 Anything V5，二次元选 Counterfeit。放入 models/Stable-diffusion/ 目录。

截图预留位置：Stable Diffusion - 下载基础模型
步骤 4

使用 LoRA 精细控制风格

LoRA 是轻量化风格模型，放入 models/Lora/ 目录后，在提示词中用 <lora:文件名:权重> 调用。权重范围 0-1，推荐 0.6-0.8，过高会导致图像崩坏。

截图预留位置：Stable Diffusion - 使用 LoRA 精细控制风格

实用小建议

  • Negative Prompt（负向提示词）同样重要，常用：ugly, blurry, extra fingers, watermark, text, bad anatomy。
  • CFG Scale（提示词遵循度）推荐 7-12，过高图像过曝，过低细节模糊。Steps 推荐 20-30。
