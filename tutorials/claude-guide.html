适合谁看？

Claude 是 Anthropic 出品的 AI 助手，以超长上下文（最高 200K tokens）、安全可靠和出色的写作与代码质量著称。它尤其擅长处理长文档、学术写作和需要深度推理的任务。

步骤 1

访问并注册

打开 claude.ai，用 Google 或邮箱注册免费账号。免费版每天可发送一定数量的消息，Claude 3 Sonnet 免费可用。

截图预留位置：Claude - 访问并注册
步骤 2

上传超长文档

Claude 支持上传 PDF、Word、TXT 等文档，最长可处理约 150,000 字的内容。可用于合同审查、论文摘要、书籍解读等场景。

截图预留位置：Claude - 上传超长文档
步骤 3

利用 Projects 功能

Projects 允许你在一个空间内保存多轮对话、文档和自定义指令，适合长期项目（如写书、产品开发）持续使用。

截图预留位置：Claude - 利用 Projects 功能
步骤 4

善用写作与代码场景

Claude 在创意写作、学术论文、代码审查和 Debug 方面表现优异。直接粘贴代码并描述问题，它会给出详细的分析与修改建议。

截图预留位置：Claude - 善用写作与代码场景

实用小建议

  • 要求 Claude 「逐步思考」或「展示推理过程」可显著提高回答准确率。
  • Claude 比较保守，遇到敏感内容会主动说明。想要更直接的回答，可在 System prompt 中明确说明使用场景。
