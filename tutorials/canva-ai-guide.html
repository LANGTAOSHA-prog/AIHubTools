Canva AI 专为设计小白打造，让不会设计的你也能轻松做出好看的图片。把它当成一个智能助手，我们一步步做出第一张海报。

一、找到 Canva AI 的入口

打开 Canva 网页版，进入任意一个设计页面。在左侧工具栏找到 “应用 (Apps)”，搜索或直接点击 Magic Media。这里就是 AI 文生图的核心功能区。如果想让 AI 帮你写文案，可以找到 Magic Write 功能。

二、掌握3个最核心的 AI 功能

1. 文生图（Magic Media）：一句话生成图片

在 Magic Media 的输入框里，用一句话描述你想要的画面，越具体越好。例如：

生成一张粉色办公桌，桌上有一杯咖啡和绿色植物，阳光从窗户照进来，温馨简约风格

2. 文字效果（Text Effects）：让标题更亮眼

在画布上添加文字，比如输入“SALE”，然后在上方工具栏找到“效果”，尝试选择“描边”或“霓虹灯”样式，能让你的标题立刻变得醒目。

3. 魔法设计（Magic Design）：自动排版布局

如果你对排版没有头绪，可以上传一张图片，Canva 的“魔法设计”会自动为你生成多种包含这张图片的版式模板，供你选择。

三、实战项目：制作周末咖啡促销海报

现在，我们整合运用以上功能，制作一张简单的“周末咖啡促销”海报。

  1. 生成背景图： 在 Magic Media 中输入提示词，生成一张有咖啡馆氛围的背景图。
    温暖咖啡馆环境，木质桌面，咖啡杯特写，阳光照射，商业海报背景，干净留白
  2. 添加标题： 添加主标题“周末咖啡买一送一”，并使用文字效果让它更突出。
  3. 添加说明文字： 用小号字体添加“本周六至周日限定”和“全场咖啡第二杯免费”等活动细则。
  4. 预留行动号召： 在右下角画一个方框，模拟二维码位置，并配上文字“扫码领取优惠券”。

四、新手练习建议

刚开始练习时，不要追求复杂的商业设计。建议从制作一张生活化的社交媒体配图开始，例如小红书的帖子图。这类主题更贴近生活，AI 生成的效果往往也更自然。

文案：周末一个人喝咖啡，也是一种治愈
图片提示词：温暖咖啡馆靠窗座位，阳光洒落，治愈风格，适合小红书封面

五、常见问题与解答 (FAQ)

Canva AI 是免费的吗？
Canva 提供了免费版本，其中包含一定额度的 AI 功能免费使用次数（例如 Magic Media 的文生图点数）。对于重度用户，订阅 Canva Pro 可以解锁无限次数和更多高级功能。
AI 生成的图片有水印吗？
通过 Magic Media 生成的图片是无水印的。但如果你在设计中使用了 Canva 素材库里的 Pro 级别（有皇冠标识）的模板或素材，导出时可能会提示需要付费。
为什么我的提示词没有效果？
提示词要尽量具体。不要只说“一张好看的图”，要描述清楚主体、环境、风格、颜色等细节。可以参考下方的提示词模板。
Magic Write 和 Magic Media 有什么区别？
Magic Media 是用来生成和编辑图片的 AI 功能（文生图）。Magic Write 是用来生成和润色文字的 AI 功能，可以帮你写标题、文案、甚至文章大纲。

六、高效提示词模板

1. 场景/主体 + 风格 + 用途：温暖咖啡馆环境，木质桌面，咖啡杯特写，商业广告风格，干净留白
2. 风格 + 氛围 + 用途：极简生活方式摄影，柔和阳光，治愈感，适合小红书封面
3. 主体 + 背景 + 风格：高端化妆品展示台，柔光背景，电商广告风格，精致干净
4. 风格 + 元素 + 用途：蓝色科技感背景，未来感界面元素，企业宣传海报风格
5. 场景 + 元素 + 风格：节日活动海报背景，灯光装饰，礼物盒，高质量商业海报风格
