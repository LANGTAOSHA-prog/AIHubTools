适合谁看？

Cursor 是专为 AI 时代设计的代码编辑器，基于 VS Code 构建，内置 GPT-4、Claude 等多种顶级模型。与 Copilot 不同，Cursor 可以理解整个代码库上下文，实现更大范围的代码修改和重构。

步骤 1

下载与安装

前往 cursor.sh 下载对应系统版本，安装后可一键导入 VS Code 的插件和设置。免费版每月可使用 2000 次 AI 补全 + 50 次 GPT-4/Claude 请求。

截图预留位置：Cursor - 下载与安装
步骤 2

掌握三大核心功能

① Tab 补全（同 Copilot）；② Cmd+K 内联编辑（选中代码后用自然语言指令修改）；③ Cmd+L 打开 Chat 侧边栏（可与整个代码库对话）。这三个功能覆盖 80% 的使用场景。

截图预留位置：Cursor - 掌握三大核心功能
步骤 3

@Codebase 全库问答

在 Chat 中输入 @Codebase，Cursor 会自动索引你的整个项目，回答「这个项目的认证逻辑在哪里？」或「我需要在哪里添加一个新的 API 路由？」等架构级问题。

截图预留位置：Cursor - @Codebase 全库问答
步骤 4

Composer 多文件重构

按 Cmd+Shift+I 打开 Composer，可以下达「把所有组件从 class 改为 hooks 写法」这类跨多个文件的大型重构任务，Cursor 会生成 diff 供你审查后一键应用。

截图预留位置：Cursor - Composer 多文件重构

实用小建议

  • 项目根目录创建 .cursorrules 文件，写入你的项目规范（语言、框架、代码风格），Cursor 会全程遵守这些规则。
  • Chat 里 @ 可以引用特定文件、函数、文档，让 AI 回答更精准。如 @utils.ts 或 @README.md。
