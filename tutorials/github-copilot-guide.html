适合谁看？

GitHub Copilot 是目前最成熟的 AI 编程助手，由 GitHub 和 OpenAI 联合开发。深度集成 VS Code、JetBrains 等主流 IDE，实时提供代码补全、函数生成和错误解释功能。学生可免费开通。

步骤 1

开通 Copilot

登录 GitHub 后，前往 github.com/features/copilot 订阅（$10/月，学生免费）。学生需在 GitHub Education 认证学生身份后可免费激活。

截图预留位置：GitHub Copilot - 开通 Copilot
步骤 2

安装 VS Code 插件

在 VS Code 扩展市场搜索「GitHub Copilot」，安装后用 GitHub 账号登录授权，即可开始使用自动补全功能。

截图预留位置：GitHub Copilot - 安装 VS Code 插件
步骤 3

使用 Tab 接受补全

Copilot 会以灰色文字实时显示建议代码。按 Tab 接受全部，按 Alt+] 切换下一个建议，按 Esc 忽略。鼠标悬停在建议上可查看备选方案。

截图预留位置：GitHub Copilot - 使用 Tab 接受补全
步骤 4

使用 Copilot Chat

按 Ctrl+I 唤出 Inline Chat，直接在代码中提问或下达指令（如「优化这段代码」「为这个函数写单元测试」）。侧边栏 Chat 面板可进行多轮对话式编程问答。

截图预留位置：GitHub Copilot - 使用 Copilot Chat

实用小建议

  • 写清楚函数名和注释，Copilot 的补全质量会大幅提升。好的命名 = 好的 AI 提示词。
  • 遇到难以解决的 Bug，在 Chat 面板选中代码后输入 /fix，Copilot 会自动诊断并给出修复方案。
